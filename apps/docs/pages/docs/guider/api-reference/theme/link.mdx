# `link()`

A standard link for your navigation items. You can also nest links.

## Example

```tsx title="theme.config.tsx"
export default defineTheme({
  directories: [
    directory("dir-a", {
      sidebar: [
        link("Home", "/home"),
        link.nested("Guides", [
          link("Guide A", "/guides/a"),
          link("Guide B", "/guides/b"),
        ]),
      ]
    })
  ],
})
```


## Reference - Link parameters

```tsx
function link(title, to, options);
```

<Field title="title" type="string" required>
  The title of the link.
</Field>

<Field title="to" type="string" required>
  Where the link goes to. For example: `/docs/guides/guide-a`.
</Field>

<Field title="options" type="ExtraLinkOptions">
  Extra options for this link. All fields are optional.

  <Field.Properties>
    <Field title="icon" type="string">
      The icon to display in the navigation item, choose from [icones](https://icones.js.org/).
    </Field>
    <Field title="newTab" type="boolean">
      If clicking the link should open a new tab or not.
    </Field>
    <Field title="exact" type="boolean">
      If the active page check should check the exact page and not its child routes. Useful in the case where you have a homepage in navigation.
    </Field>
    <Field title="style" type="string">
      The display style of the link, can be `default` or `star`.
    </Field>
  </Field.Properties>
</Field>
